We will add one more process to this stage.

We will create a table that will track the customer coordination between a user and the handling process to the customer.
This table will be similar to the TaskPostHandling table, it will be linked to a task, have it's own enum states, have a creation and update columns and the polymorphic history table will also hold the state transtions of this table.

This table will live in task tables folder, and it's state enums will be: pending, coordinating, completed.
